Project Overview

DashboardWeb Development

SyncSpace is an advanced platform tailored to seamlessly manage physical and hybrid workspaces.

. Integrated 100% with Microsoft 365, it redefines space management by allowing organizations to digitalize interactive floor plans, manage workstation bookings, optimize parking allocations (like EV charging spots), and seamlessly coordinate hybrid employee flows.

Scope of Work

The core objective of this project was to design the user experience and user interface for the SyncSpace client-side Microsoft Teams plugin. Because the system relies heavily on complex integration strings and backend architecture to successfully link a company's Microsoft 365 environment with SyncSpace's physical booking features, the primary UX challenge was mapping out an effortless installation, configuration, and setup flow for end-users.

  • End-to-End UX Mapping & Wireframing: Collaborated closely with a design peer to map out the complete user journey, architecture, and structural wireframes ensuring all Microsoft Teams touchpoints were intuitive.
  • Onboarding & Configuration UX Strategy:Conducted deep-dive workflow research to dissect and map out the complex background connections required for the client-side configuration. Designed custom step-by-step installation guides and setup dashboards that stripped away technical complexity.
  • UI Design & Developer Handoff:Engineered the final high-fidelity User Interface (UI), maintaining brand consistency with the main platform. Fully mapped out interaction behaviors and edge cases to ensure a frictionless handoff to the engineering team.
  • Web Development:Independently developed, tested, and deployed the official responsive website, ensuring clean code, fast performance, and accurate brand representation.

Final Outcome

The project culminated in a highly successful dual delivery: a polished, developer-ready interface for the Microsoft Teams plugin and a fully functional, live website.

By introducing guided setup panels, we successfully transformed a dense, technical backend configuration into a frictionless onboarding experience. Coupled with the newly developed web platform, SyncSpace now offers users a unified, intuitive, and highly professional entry point to hybrid workspace management.
